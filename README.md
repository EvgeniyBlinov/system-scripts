# system-scripts
My system scripts


## Install

```sh
git clone https://github.com/EvgeniyBlinov/system-scripts /usr/local/share/system-scripts &&
cd /usr/local/share/system-scripts &&
ln -s /usr/local/share/system-scripts/bin/* /usr/local/bin/
```
